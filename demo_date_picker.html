For more details on how to use this toolkit, refer to the following link: Persian Date Picker

Apart from the instructions given in the link above you can take advantage of the date_picker_initialize.js to easily customize the Pashto Datepicker.


Without Time Picker

To use only the datepicker to select year, month, day

e.g.

To Use: <html> <meta charset="utf-8"/> <script src="lib/jquery.js"></script> <link rel="stylesheet" href="dist/css/persian-datepicker-0.4.5.css"/> <link href="lib/bootstrap/css/bootstrap.min.css" rel="stylesheet" type="text/css"/> <script type="text/javascript" src="lib/bootstrap/js/bootstrap.min.js"></script> <link href="dist/css/persian-datepicker-0.4.5.css" rel="stylesheet" type="text/css"/> <script src="lib/persian-date.js"></script> <script type="text/javascript" src="dist/js/persian-datepicker-0.4.5.js"></script> <script type="text/javascript" src="dist/js/date_picker_initialize.js"></script> </html> <body> <input type="text" class="pashto_datepicker"/> </body>
